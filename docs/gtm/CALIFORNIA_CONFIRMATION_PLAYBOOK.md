# California Confirmation Playbook

Use this after the California source-refresh pass to tighten the remaining high-value unknowns without guessing.

## Goal

Convert the remaining trust-critical unknowns in the live California 8 from:

- blank because we could not defend them from public sources

to:

- therapist-confirmed with a date, or
- editor-verified if the therapist provides a source we can independently confirm

## What Belongs Here

Only ask specialists to confirm fields that are:

- high-leverage for user trust
- hard to source reliably in public
- likely to change over time

For the current California launch set, that mainly means:

- estimated wait time
- bipolar-specific years of experience
- insurance or superbill reality
- telehealth scope if still unclear
- license number only where public source confidence is weak or conflicting

## Operating Rule

Do not fill unknowns because they seem likely.

Only upgrade a field when one of these becomes true:

1. The therapist confirms it directly.
2. The therapist sends a source we can independently verify.
3. We find a strong public source that clearly states it.

If none of those happen, leave the field blank or keep it marked as needing re-confirmation.

## Suggested Confirmation Prompt Shape

For each therapist, keep the ask short and operational:

1. Are you currently accepting new bipolar clients?
2. What is your typical wait time right now?
3. About how many years have you worked specifically with bipolar-spectrum clients?
4. Do you accept insurance, submit out-of-network claims, or provide superbills?
5. Are the listed telehealth and in-person options still accurate?

## Success Standard

A California launch profile is strong when:

- license and location are source-backed
- telehealth / in-person scope is reliable
- insurance stance is honest
- first-contact flow is clear
- freshness is recent enough to trust
- remaining unknowns are small and explicit, not hidden
